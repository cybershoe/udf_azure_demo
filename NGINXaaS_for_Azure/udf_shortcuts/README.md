udf_shortcuts
=============

Convenience features to make working in UDF easier

Input variables:
----------------

| Variable            | Type     | Description                                        |
| ------------------- | -------- | -------------------------------------------------- |
| ip_address                | string   | IP address of the NGINXaaS deployment  |
| desktop_path | string   | Path to the user's Desktop directory                     |

Required providers:
-------------------

- local ~> 2.4
